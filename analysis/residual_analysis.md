# Residual Analysis

The final multiple regression model was used to predict sales.  
Residuals = Actual Sales – Predicted Sales.  

Top positive residuals: some Mall and Airport stores sold more than predicted → model under-predicts high-traffic locations.  
Top negative residuals: some Residential and High Street stores sold less than predicted → model over-predicts weaker stores.  

This means the model fits well overall but misses store-specific effects. Leadership should investigate outliers to learn best practices or fix problem areas.
